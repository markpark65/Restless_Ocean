# 🔱 Restless Ocean

> **"잊혀진 심해의 유적, 굶주린 심연의 공포. 당신은 살아남아 비밀을 밝혀낼 것인가?"**

'Restless Ocean'은 터미널 기반의 1인칭 심해 탐사 생존 RPG입니다. 플레이어는 탐사 대원이 되어, 미스터리한 심해 유적을 탐험하고, 기괴한 심해 몬스터들과 전투하며, 유적의 비밀을 밝혀내야 합니다. 한정된 산소, 배터리, 그리고 서서히 조여오는 수압을 관리하며 심연의 끝에 도달하세요.

---

## 🛠️ 설치 및 실행 방법 (How to Install & Run)

이 프로젝트는 **C++17** 표준을 기반으로 작성되었습니다. 원활한 빌드 및 실행을 위해 아래의 요구사항이 필요합니다.

### 📋 사전 요구 사항 (Prerequisites)

* **컴파일러**: C++17 이상을 지원하는 컴파일러 (Visual Studio 2019/2022 권장, GCC/Clang 7 이상)
* **개발 환경 (IDE)**: Visual Studio, VS Code, 또는 C++ 개발이 가능한 텍스트 에디터

### 🚀 소스 코드에서 빌드 및 실행하기 (Building from Source)

#### A. Visual Studio (Windows)

1.  이 저장소를 복사(Clone)합니다.
    ```bash
    git clone [https://github.com/](https://github.com/)[GitHub 사용자명]/RestlessOcean.git
    ```
2.  Visual Studio를 실행하고 **"솔루션 또는 프로젝트 열기"**를 선택한 후, 프로젝트 폴더의 `RestlessOcean.sln` 파일을 엽니다.
3.  상단 도구 모음에서 빌드 구성을 **`Release`**로, 플랫폼을 **`x64`**로 설정합니다.
4.  메뉴에서 `빌드` -> **`솔루션 빌드`** (또는 `Ctrl + Shift + B`)를 선택하여 컴파일합니다.
5.  빌드가 완료되면 `F5` 키를 누르거나, 출력 폴더(`x64/Release/`)에 생성된 `RestlessOcean.exe` 파일을 실행합니다.

#### B. GCC/Clang (Linux/macOS)

1.  터미널을 열고 저장소를 복사(Clone)합니다.
2.  프로젝트 소스 파일이 있는 폴더로 이동합니다.
3.  아래 명령어를 입력하여 컴파일합니다.
    ```bash
    g++ -std=c++17 *.cpp -o RestlessOcean
    ```
    * *(참고: 만약 헤더 파일이 `include/` 폴더에 있다면 `-Iinclude` 옵션을 추가하세요.)*
4.  컴파일된 프로그램을 실행합니다.
    ```bash
    ./RestlessOcean
    ```

## 🕹️ 조작 방법 (Controls)

본 게임은 터미널 기반의 텍스트 메뉴 방식으로 동작하며, 모든 조작은 **숫자 입력**으로 이루어집니다.

1.  터미널에 표시된 메뉴 중 원하는 행동에 해당하는 **숫자를 입력**합니다.
2.  **`Enter`** 키를 눌러 행동을 확정합니다.

```text
[SeaCave - 1 / 5 단계]
================================
행동을 선택하세요:
1. 일반 공격
2. 스킬 사용 (배터리 20% 소모)
3. 아이템 사용
4. 도망치기
> [이곳에 숫자를 입력하세요]
```

## 🌌 게임 시스템 가이드 (Game System Guide)

성공적인 심해 탐사를 위해 대원 mark의 생존 수치와 주변 환경을 잘 관리해야 합니다.

### 1. 주요 생존 수치 (Status)

* **O2 (산소)**: 🔵 전투 종료 시마다 **10%**씩 소모됩니다. 0%가 되면 체력이 매 턴마다 감소합니다. 로비에서 회복하거나 산소 회복 아이템을 사용하세요.
* **Battery (배터리)**: ⚡ 강력한 '스킬'을 사용하는 데 필요한 고에너지 자원입니다. 스킬마다 다른 소모값(기본 20%)을 가지며, 로비에서 회복하거나 충전 아이템을 사용하여 완충할 수 있습니다. 무기 사용 시에는 소모되지 않습니다.
* **Pressure (수압)**: 🔴 던전 깊이 들어갈수록 상승합니다. 100%에 도달하면 과도한 수압으로 인해 **속도(Speed)가 크게 감소**하는 디버프를 받습니다. 로비에서 초기화 가능하며, 도망칠 경우 0%로 초기화되지만 산소는 소모됩니다.
* **HP (체력)**: ❤️ 플레이어의 생명력입니다. 0이 되면 게임 오버. 로비나 아이템으로 회복할 수 있습니다.

### 2. 탐사 및 전투 (Exploration & Battle)

* **유적 탐사 단계**: 각 유적은 5단계로 구성되어 있습니다.
    * **1 ~ 4단계**: 해당 맵 테마에 맞는 일반 몬스터가 출현합니다.
    * **5단계 (보스)**: 강력한 보스 몬스터가 출현합니다. 보스를 처치해야 유물을 획득할 수 있습니다.
* **속도 (Speed)**: 전투에서 행동 우선순위를 결정합니다. 플레이어가 몬스터보다 빠르면 먼저 공격합니다.

### 3. 유적 (Artifacts)

* 이 게임의 최종 목표입니다. 총 **3개**의 각기 다른 유적을 모두 수집하면 심해의 비밀을 밝혀내며 게임 승리합니다.
